# Business Performance Analysis – Sales, Profit & Returns

## Overview
This project analyzes sales performance, profitability, and return patterns using Excel.
The objective is to understand where the business is generating profit, where it is losing money, and why.

---

## Dataset
The analysis is based on retail sales data containing order-level details such as sales, profit,
product category, region, and return information.

---

## Excel Sheet Structure

**Orders**
Contains transaction-level sales data used as the primary source for analysis.

**Returns**
Used to identify returned orders and analyze their impact on profitability.

**Users**
Provides additional customer and regional details to support segmentation.

**Pivots**
Contains pivot tables used to calculate KPIs and prepare dashboard visuals.

**Dashboard**
The final interactive dashboard presenting key metrics, trends, insights, and recommendations.

---

## Key Insights
- High sales volume does not always result in high profitability
- Returns and discounting significantly impact margins
- A small number of regions contribute most of the profit
- Certain categories consistently generate losses

---

## Business Recommendations
- Review discount strategies in loss.
-making categories
- Investigate high return rates and address root causes
- Focus growth efforts on high-margin regions
- Re-evaluate pricing strategies where losses persist

---

## Tools Used
- Microsoft Excel
- Pivot Tables and Charts
- Slicers
- Calculated fields
